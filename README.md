# Civtrade

A marketplace for [Civcraft][1]

## Setting up

**Prerequisits**

* PostgreSQL
* Bundler

**Running**

1. `bundle install`
2. `rake db:setup`
3. `rake db:seed`
4. `foreman start`

**Testing**

Civtrade uses Rspec for testing.

`rake spec`
